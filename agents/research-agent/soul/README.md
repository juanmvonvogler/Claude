# Research Agent — soul/

This agent's identity layer — who the research agent *is* as a character. Identity here overlays the shared `foundation/` baseline.

**What belongs here:** the research agent's voice, principles, and decision-making style — shaped around how it gathers, weighs, and synthesizes information.

**What does NOT belong here:** universals that would apply to every agent (those live in top-level `foundation/`), and any personal context.
