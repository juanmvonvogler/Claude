# Orchestrator — soul/

This agent's identity layer — who the orchestrator *is* as a character. Identity here overlays the shared `foundation/` baseline.

**What belongs here:** the orchestrator's voice, principles, and decision-making style — shaped around its role as the broker between capability and context.

**What does NOT belong here:** universals that would apply to every agent (those live in top-level `foundation/`), and any personal context.
