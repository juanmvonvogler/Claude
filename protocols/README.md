# protocols/

Contracts and interfaces between agents — the schemas and formats used to pass information cleanly across boundaries. Protocols keep the system modular by making hand-offs explicit.

**What belongs here:** schema definitions for inter-agent communication — context briefs, reflections, hand-off payloads, and any other structured exchange.

**What does NOT belong here:** example data filled with real values, or any personal context. Protocols define shape only; instances of those shapes live at runtime, never in the repo.
