# foundation/

The thin universal layer — the substrate every agent inherits regardless of character. Kept deliberately small; agent identity lives per-agent in `agents/<name>/soul/`.

**What belongs here:** baseline behaviors true for every agent — guardrails against fabrication, norms around uncertainty, rules of engagement that apply whether the agent is a researcher, a builder, or an orchestrator.

**What does NOT belong here:** anything that would vary by agent character — tone, voice, role-specific principles (those belong in that agent's `soul/`) — and any personal context. If adding something here would force one of the agents to partially undo it, it doesn't belong here.
